#Mjctechguy's Dotfiles
#### Included dotfiles
- sway
- waybar
- mako
- qt5ct
- gtk-3.0
- alacritty
- fish
- ncspot
- micro
- rofi

**Setting up config files**
```
mkdir ~/.dotfiles
cd ~/.dotfiles
git clone https://github.com/mjctechguy/dotfiles.git
chmod +x setup
./setup
```
